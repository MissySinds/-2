# Missy - Лендинг магазина косметики

Адаптивный лендинг для магазина косметики, разработанный на Bootstrap 5 в рамках практического задания по курсу "Программная верстка в дизайне".

## О проекте

Лендинг включает все необходимые секции:
- Шапка с навигацией
- Hero-секция
- О магазине
- Каталог растений
- Рассылка
- FAQ
- Контакты
- Подвал

## Технологии

- HTML5
- CSS3
- Bootstrap 5.3
- JavaScript
- Bootstrap Icons

## Функциональность

### Обязательные компоненты Bootstrap:
- ✅ Адаптивная сетка
- ✅ Форма с валидацией
- ✅ Слайдер (карусель)
- ✅ Модальное окно
- ✅ Бургер-меню
- ✅ Табы
- ✅ Аккордеон

### Дополнительные компоненты:
- Карточки товаров
- Плавная навигация
- Адаптивный дизайн

## Адаптивность

Контрольные точки:
- 1440px
- 1024px  
- 768px
